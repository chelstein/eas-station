# Debug Scripts

This directory contains debugging and testing scripts for development purposes.

## Files

- `debug_airspy.py` - Airspy SDR hardware debugging
- `debug_redis_metrics.py` - Redis metrics and connection debugging
- `test_sdr_direct.py` - Direct SDR hardware testing

## Usage

These scripts are for development and troubleshooting only. They are not required for production deployment.

Run from the project root:
```bash
python scripts/debug/debug_airspy.py
```
