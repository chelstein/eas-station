# Maintenance Scripts

This directory contains database maintenance and system repair scripts.

## Files

- `apply_db_fixes.py` - Applies database schema fixes and corrections
- `apply_driver_fix.py` - Applies SDR driver fixes

## Usage

These scripts should be run with caution, typically only when instructed by documentation or support.

**Always backup your database before running maintenance scripts.**

Run from the project root:
```bash
python scripts/maintenance/apply_db_fixes.py
```
