EAS Decoder Monitor

Configure audio monitoring for the EAS decoder pipeline

Decoder Monitor Settings

What is this? The EAS decoder monitor provides a way to listen to the actual 16 kHz resampled audio fed to the SAME decoder. This helps verify that:
  • Audio is being correctly resampled to 16 kHz for decoding
  • Audio quality is sufficient for SAME header detection
  • No distortion or clipping in the decoder input path
When enabled, creates a monitoring stream of the 16 kHz audio fed to the EAS decoder. This stream can be accessed on the Audio Monitoring page.
Name for the decoder monitoring stream (appears in audio sources list).

Technical Details

Sample Rate

The EAS decoder expects audio at exactly 16,000 Hz (16 kHz) for SAME header detection.

Resampling

Audio sources typically run at 32-48 kHz. The ResamplingBroadcastAdapter converts this to 16 kHz using linear interpolation.

Monitoring Stream

The decoder monitor taps into the resampled audio stream, providing a separate monitoring output at 16 kHz.

Bandwidth

At 16 kHz mono, the uncompressed stream uses ~256 kbps (16-bit PCM).
