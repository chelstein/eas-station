EAS Decoder Monitor

Configure audio monitoring for the EAS decoder pipeline and inject test signals

Decoder Monitor Settings

What is this? The EAS decoder monitor streams the exact 16 kHz audio each source sends to the SAME decoder. When enabled, each running audio source gets its own Icecast mount point so you can listen to what the decoder hears for that channel.
  • Audio is correctly resampled to 16 kHz for SAME decoding
  • Audio quality is sufficient for header detection
  • No distortion or clipping in the decoder input path
When enabled, creates a per-source 16 kHz Icecast monitoring stream for every running audio source.
Prefix for the per-source mount points. A source named my-receiver will appear at /{prefix}-my-receiver.mp3 on the Icecast server (port 8000).

Inject Test EAS Signal

Decoder pipeline test. Injects a synthetic SAME Required Weekly Test (RWT) signal directly into the 16 kHz EAS broadcast queue. The signal will be processed by the live EAS decoder exactly as if it arrived from a real receiver — you can monitor the result in the Active Alerts page and in any active EAS ingest Icecast streams.
Leave blank to inject into the first running source found.

Mount Point Reference

Per-Source Streams

Each running audio source gets its own monitoring mount point:
/{prefix}-{source-name}.mp3
Access directly on Icecast (port 8000) or through the nginx proxy.

Sample Rate

All decoder monitor streams run at 16,000 Hz mono — exactly what the SAME decoder processes.

Resampling

Sources typically run at 32–48 kHz. The ResamplingBroadcastAdapter converts to 16 kHz before publishing to the EAS queue.

Bandwidth

Each stream uses ~48 kbps (MP3, 16 kHz mono).

Test Signal Details

The injected signal is a standards-compliant SAME message:

  • Event: RWT (Required Weekly Test)
  • Originator: EAS
  • Location: 000000 (all areas)
  • Duration: 15 minutes
  • 3× header bursts + EOM per FCC §11.31

The signal bypasses the actual radio receiver and is injected at the 16 kHz resampled stage, so it tests the decoder and all downstream pipeline stages.
