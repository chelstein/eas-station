Testing Instructions
How to Test Themes:
  1. Click the "Change Theme" button above
  2. Select each theme from the modal
  3. Verify that:
    • Logo adapts to theme colors
    • All card headers are readable
    • Form controls (checkboxes, switches) are visible
    • Form labels are clearly readable
    • Button text has good contrast
    • Badge text is readable
    • All body text is comfortable to read
  4. Report any readability issues found
